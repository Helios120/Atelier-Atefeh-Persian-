## Mise en ligne GitHub Pages

1. Créer un repository nommé atelier-atefeh-persian.
2. Mettre le repository en Public.
3. Ajouter tous les fichiers à la racine.
4. Créer le dossier assets.
5. Ajouter les images dans assets.
6. Aller dans Settings puis Pages.
7. Choisir Deploy from a branch.
8. Branch : main.
9. Folder : /root.
10. Cliquer sur Save.

GitHub donnera une URL du type :

https://votre-utilisateur.github.io/atelier-atefeh-persian/

Remplacer cette URL dans :

- index.html
- portfolio.html
- services.html
- a-propos.html
- contact.html
- robots.txt
- sitemap.xml
